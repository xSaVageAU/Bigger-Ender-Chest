# Bigger Ender Chest

A Fabric mod for Minecraft 26.1+ that expands the capacity of the Ender Chest.

## Features
- Increased Ender Chest storage space.
- Configurable capacity (planned).

## Setup
For setup instructions, please see the [Fabric Documentation](https://docs.fabricmc.net/develop/).
